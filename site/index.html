Claim your name View the registry on GitHub

What you get

A subdomain

yourname.devis.im, pointed at GitHub Pages, Vercel, Netlify, a raw server, or a redirect. HTTPS included automatically.

An email alias (optional)

yourname@devis.im forwards to your private inbox. Your real address is never exposed — and never stored in the public repo.

How it works

  1. Open a pull request adding one file: domains/yourname.json.
  2. A maintainer reviews it. If the name is free and the request looks fine, it's merged.
  3. On merge, automation creates your DNS record + HTTPS within minutes.
  4. Want email? Set "email": { "enabled": true }, then add your forwarding address through a private form after merge and click the verification link.

The whole file

{
  "$schema": "../schema.json",
  "owner": { "github": "your-github-username" },
  "record": { "CNAME": "your-github-username.github.io" },
  "email": { "enabled": true }
}

The filename is the name you claim. Use CNAME or A/AAAA, plus TXT and URL records. Full format and reserved names are in the README.
