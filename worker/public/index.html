Set up your name@devis.im forwarding

Enter the name you already claimed and the inbox you want mail forwarded to. When you click Activate you'll sign in with GitHub in a popup — this proves the name is yours; you can only set up forwarding for a name your own GitHub account claimed. If the inbox is new to devis.im, Cloudflare emails it a verification link — click it, then submit this form once more to finish. Your address is sent privately and is never stored in the public registry.
